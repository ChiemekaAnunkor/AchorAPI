Excel Generator

Quick tip: only takes an array of objects, nested objects or nested arrays would return inconsistent data

How to get Generate Excel Document of Your Data
  • Unput your html into the broswer input
  • Click populate data (make sure it is the correct data)
  • Click on the generate fuction

Using this feature to convert your data into html docs, this will only work on flatted arrays. If you have data is many times nested, it might work but the data will not be sorted properly in the excel tables. The objects inside can have as many key value pairs as you want and it will still work for you
